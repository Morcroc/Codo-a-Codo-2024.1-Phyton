Uso de Favicon

Si bien se puede colocar archivos de imagen con extensión .png como Favicon, también se pueden convertir a archivos con extensión .ico

Pasos para colocarlo (imagen .png)

  1. Buscar o crear una imagen .png
  2. En el head del documento HTML colocar la referencia con: link rel="icon" href="nombredelarchivo.ico"

Pasos para colocarlo (imagen .ico)

  1. Buscar o crear una imagen .png
  2. Ingresar en https://convertico.com/
  3. Seleccionar el archivo, convertirtlo y descargarlo
  4. En el head del documento HTML colocar la referencia con: link rel="icon" href="nombredelarchivo.ico"
